# Sistema de Gestão para Lanchonete

Projeto desenvolvido para estudo utilizando uma arquitetura Full Stack.

## Tecnologias

- React
- Node.js
- Express
- PostgreSQL
- Git
- GitHub

## Objetivo

Construir um sistema completo de gerenciamento de pedidos, clientes, produtos e estoque.

## Funcionalidades previstas

- Login
- Cadastro de clientes
- Cadastro de produtos
- Cadastro de categorias
- Controle de estoque
- Pedidos
- Dashboard
